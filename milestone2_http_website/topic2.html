SPDY → HTTP/2 → HTTP/3

Google's SPDY introduced multiplexing, header compression, and server push. HTTP/2 standardized these ideas using a binary framing layer. HTTP/3 replaced TCP with QUIC (on top of UDP) to eliminate TCP head-of-line blocking and support faster connection migration.

QUIC advantages

  • Independent multiplexed streams
  • Faster connection setup
  • Connection migration between networks (e.g., Wi-Fi→cellular)
