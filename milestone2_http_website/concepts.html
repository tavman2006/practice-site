5–7 Key Concepts

HTTP
Application-layer protocol for web communication.
TCP
Ordered, reliable transport protocol used by older HTTP versions.
QUIC
UDP-based transport protocol with independent streams used by HTTP/3.
HTTPS
HTTP over TLS — encrypts requests and responses.
SPDY
Google's experimental protocol that inspired HTTP/2.
