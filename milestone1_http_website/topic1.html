History of HTTP

HTTP was first created by Tim Berners-Lee at CERN in 1989 to allow scientists to share documents globally. The earliest version, HTTP/0.9, only supported one command, GET, which fetched simple text pages.

HTTP/1.0 introduced headers and support for multimedia content, while HTTP/1.1 improved efficiency with persistent connections, caching, and chunked transfers, laying the foundation for modern web browsing.
